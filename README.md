# 千葉カケルくんシューティングげーむ

東方風のオリジナル縦スクロールシューティングです。Canvas で動くWebゲームとして実装しています。

## 特徴

- WASD 移動、Space ショット
- 通常モード
- C でスペルカード発動
- ライフは 2
- ライフ減少時にボムは 3 まで回復
- 縦長のステージ画面
- wave1: 雑魚Aを15隊
- wave2: 雑魚Bを15隊
- wave3-1 / wave3-2: 中ボス 2連戦
- wave4: 雑魚A + 雑魚B の混成
- ボス会話 → BOSS_1 の 6 フェイズ戦
- ボス前会話あり
- 大ボスは 6 種類の弾幕パターンを持つ
- プレイ時間は約 15 分を想定

## 起動

このプロジェクトは静的ファイルのみで動きます。`index.html` をブラウザで開いてください。

## 変更の入口

- ゲームロジック: `main.js`
- 見た目: `styles.css`
- 画面構成: `index.html`
